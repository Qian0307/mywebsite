✦ 星辰術士・雙月 ✦

我是 雙月 —— 一位在星光與知識之間穿梭的學者型術士。
喜歡探索人類心靈的奧秘，也熱衷於科技魔法的應用，把所有混亂整理成溫柔的秩序。

目前修行的主法術：

  • 心理學心靈術（NCKU 心理系・神經科學 / 感知覺 / 統計）
  • 數位心理健康（MindTrace・Digital Phenotyping × Just-in-time Intervention）
  • 程式魔導（JavaScript / Python / 自動化）
  • 觀測術（Notion 系統、n8n Automatons、資料整理）
  • 語言修煉（法語 DELF 修行中・英語 TOEIC 備考）

我的魔力來源是「理解」。希望某一天能創造出屬於自己的「星辰魔法書」—— 一份結合知識、科技與魔法的完整宇宙。
